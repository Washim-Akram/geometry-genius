What is DOM? Purpose of DOM ?

DOM Stands for Document Object Model. In javaScript everything considered as an objects.In HTML, Layout of website called a Document. When a Document is loaded in a web browser, JavaScript converts the document into an object.
The purpose of DOM is that access all the html elements. Create, Read, Update, Delete html elements from the document object.Using the DOM, we can change the whole html document.

How to select element from DOM?

There are many ways to select element from DOM. They are called Method. These methods are self explanatory. Mostly used/familiar methods are:

  1. getElementsByTagName(): Get elements from DOM by tag name.
  2. getElementById(): Get an element from DOM by an Id.
  3. getElementsByClassName(): Get elements from DOM by class name.
  4. querySelector(): Get the first element from the DOM by CSS Selector.
  5. querySelectorAll(): Get all elements from the DOM by CSS Selector.

What is event bubble in DOM?

When an event occurs in HTML DOM, in bubbling the trigger first reached its last element event. After that it go to the its upward/parent event continuously. In addEventListener(), event bubble is enabled by default.

What is callback function in JS? Purpose of callback function in JS.

In JavaScript, when a function pass as a parameter into another function, thats called the callback function.
When we want to execute a function after, complete execution of another function, we use callback function.
